IMAGENS

A tag HTML < img > é usada para incorporar imagens em uma página da web. Aqui está um resumo das principais características e atributos dessa tag:

Sintaxe

<img src=" ">

O atributo "src" é obrigatório e especifica o caminho para o arquivo de imagem que você deseja exibir. Você pode fornecer um URL absoluto ou um caminho relativo para a imagem.

<img src=" " alt=" ">

O atributo "alt" é opcional e fornece um texto alternativo para a imagem. Ele é usado para descrever a imagem, caso ela não possa ser exibida ou para fins de acessibilidade, ajudando pessoas com deficiência visual a entender o conteúdo da imagem.

<img src=" " width=" " right=" ">

Esses atributos são opcionais e permitem definir a largura e a altura da imagem em pixels. Eles são usados para controlar o tamanho da imagem na página.

Formatos de imagens suportadas

A tag em HTML pode exibir imagens em vários formatos, desde que o navegador seja capaz de renderizar o formato específico. Alguns dos formatos de imagem mais comuns que podem ser usados com a tag incluem:

JPEG

PNG

PNG ICO

GIF

SVG

BMP

BMP ICO

INPUTS

Os elementos <input> em HTML são utilizados para criar campos de entrada em formulários da web, permitindo que os usuários insiram dados, como texto, números, opções ou arquivos.

Tipos de inputs

Text

Number

Password

Number

Checkbox e radio

File

Submit e button

Date, time e datetime-local

Submit e button

Color

Range

Hidden

Sintaxe

<input type="text">

Este tipo de campo de entrada permite que os usuários insiram texto simples, como nomes, endereços de e-mail ou mensagens.

<input type="password">

Semelhante ao tipo "text", mas oculta o texto inserido, normalmente exibindo asteriscos ou pontos. É usado para senhas e outros dados confidenciais.

<input type="number">

Este tipo é usado para entrada numérica, permitindo que os usuários insiram números. Pode incluir controles para aumentar ou diminuir o valor.

<input type="checkbox">

É usado para criar caixas de seleção em formulários HTML. As caixas de seleção permitem que os usuários escolham entre opções múltiplas, marcando ou desmarcando uma ou várias caixas de seleção.

<input type="radio">

É usado para criar caixas de seleção em formulários HTML. As caixas de seleção permitem que os usuários escolham entre opções múltiplas, marcando ou desmarcando uma ou várias caixas de seleção

<input type="file">

Permite que os usuários escolham e enviem arquivos para o servidor, como imagens, documentos ou outros tipos de arquivos.

<input type="date">

Esse tipo de input é utilizado para coletar data.

<input type="time">

Esse tipo de input é utilizado para coletar hora.

<input type="datetime-local">

Esse tipo de input é utilizado para coletar data e hora.

<input type="color">

Permite que os usuários escolham uma cor por meio de uma paleta de cores.

<input type="range">

Cria um controle deslizante que permite aos usuários selecionar um valor em um intervalo predefinido.

<input type="hidden">

Esse tipo cria um campo oculto no formulário, que pode ser usado para armazenar dados que não devem ser visíveis para o usuário, mas são enviados para o servidor junto com o formulário.

TABELA

A tag HTML <table> é usada para criar tabelas em uma página da web. Tabelas são usadas para organizar dados em linhas e colunas, o que facilita a exibição e compreensão de informações tabulares.

Sintaxe

<table>
        <thead>
            <tr>
                <th>Nome</th>
                <th>Idade</th>
            </tr>
        </thead>
     <tbody>
        <tr>
                <td>João</td>
                <td>25</td>
        </tr>
        <tr>
                <td>Maria</td>
                <td>20</td>
        </tr>
      </tbody>
</table>
Nome Idade
João 25
Maria 30

<table>: Esta é a tag de abertura que inicia uma tabela.

<tr>: Dentro da tabela, você utiliza a tag <tr> (table row) para criar linhas. Cada linha da tabela deve ser definida dentro de um par de tags.

<th> e <td>: Dentro das linhas, você utiliza as tags <th> (table header) para criar células de cabeçalho e <td> (table conteudo) para criar células de conteúdo.

<thead>: É usada para agrupar as células de cabeçalho.

<tbody>: É usada para agrupar as células do corpo da tabela.

TEXTOS

Em HTML, os elementos de texto são usados para exibir conteúdo textual em uma página da web.

Tipos de tags de textos

h1

h2

h3

h4

h5

h6

p

strong

span

br

Sintaxe

<h1>Texto</h1>
<h2>Texto</h2>
<h3>Texto</h3>
<h4>Texto</h4>
<h5>Texto</h5>
<h6>Texto</h6>

É usada para criar um texto de nível 1 ao nível 6 sendo o mais importante o número 1 e assim por diante.

<p>Texto</p>

É usada para separar e formatar blocos de texto em parágrafos distintos. Cada vez que você utiliza essa tag, cria um novo parágrafo, o que ajuda a melhorar a legibilidade e organização do conteúdo da página.

<strong>Texto</strong>

É usada para indicar que o texto dentro dela deve ser exibido com ênfase, geralmente na forma de negrito. Essa ênfase forte é usada para destacar a importância ou o peso do texto, não apenas para fins de formatação visual, mas também para fins semânticos.

<span>Texto</span>

É usada para aplicar estilos ou scripts a partes específicas de texto em uma página da web, sem afetar a estrutura geral do documento.

<br>

É usada para forçar uma quebra de linha em um documento HTML, sem iniciar um novo parágrafo. Ela é útil quando você deseja quebrar o texto em várias linhas dentro do mesmo contexto, como endereços, poesia, ou sempre que a quebra de linha for necessária.

ÂNCORA

A âncora é criada com a tag <a> , que é usada para criar links hiperativos em páginas da web.

Links hiperativos, comumente referidos como "hiperlinks" ou simplesmente "links", são elementos em páginas da web que permitem que os usuários cliquem em um texto ou elemento para serem redirecionados para outra página da web, recurso ou localização. Eles são elementos interativos que conectam diferentes partes da web, permitindo a navegação de um local para outro.

Sintaxe

<a href=" " target="_blank">Visitar</a>

<a>Essa é a tag de abertura para criar um link hiperativo.

href O atributo "href" é obrigatório e define o URL da página ou recurso para o qual o link deve apontar. Pode ser um URL absoluto (começando com "http://" ou "https://") ou um URL relativo (referindo-se a um recurso dentro do mesmo site).

target O atributo "target" é opcional e define como o link deve ser aberto. Alguns valores comuns incluem "_blank" para abrir o link em uma nova janela ou guia, ou "_self" para abrir no mesmo quadro ou janela.

ESTRUTURA

A estrutura básica de um documento HTML (Hypertext Markup Language) consiste em uma série de elementos aninhados que descrevem a estrutura e o conteúdo de uma página da web.

Sintaxe

<!DOCTYPE html>
<html lang="pt-br">
    <head>
            <meta charset="UTF-8">
            <title> title </title>
    </head>
    <body>

    </body>
</html>

<!DOCTYPE html>: Esta declaração define a versão do HTML que está sendo usada, no caso, o HTML5.

<html lang="pt-br">: O elemento raiz que engloba todo o conteúdo da página.

<head>: Contém informações sobre a página que não são exibidas no navegador, como metadados, links para arquivos CSS e JavaScript, e o título da página.

<meta charset="UTF-8">: Define a codificação de caracteres para a página, geralmente definida como UTF-8 para suportar caracteres especiais e internacionais.

<title>: Define o título da página, que é exibido na barra de título do navegador.

<body>: Contém o conteúdo visível da página.

LISTAS

Listas em HTML são usadas para organizar e estruturar informações em formato de itens. Existem dois tipos principais de listas em HTML: listas não ordenadas e listas ordenadas.

Tipos de listas

Lista não ordenada <ul>

Lista ordenada <ol>

Sintaxe

<ul>
        <li> Item 1 </li>
        <li> Item 2 </li>
        <li> Item 3 </li>
</ul>
  • Item 1
  • Item 2
  • Item 3

O elemento <ul> é usado para criar listas não ordenadas em HTML.

Uma lista não ordenada é uma lista de itens onde a ordem dos itens não é significativa. Cada item da lista é representado pelo elemento <li> (item de lista)

Uma lista não ordenada é uma lista de itens onde a ordem dos itens não é significativa.

<ol>
        <li> Item </li>
        <li> Item </li>
        <li> Item </li>
</ol>
  1. Item
  2. Item
  3. Item

O elemento <ol> é usado para criar listas ordenadas em HTML.

É uma lista de itens onde a ordem dos itens não é significativa. Cada item da lista é representado pelo elemento <li> (item de lista)

Uma lista ordenada é uma lista de itens onde a ordem dos itens é significativa e geralmente exibida com números ou letras para indicar a sequência.
