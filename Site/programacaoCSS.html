ESTILIZAÇÃO DE TEXTOS

A estilização de textos em CSS envolve a aplicação de estilos a elementos de texto, como títulos, parágrafos, links e outros.

Sintaxe

p {
        font-family: Arial, sans-serif;
}

font-family: Define a fonte a ser usada para o texto. 

h1 {
        font-size: 24px;
}

font-size: Define o tamanho do texto.

24px: Tamanho fixo em pixels.

2rem: Um valor relativo à fonte do elemento raiz. 

strong {
            font-weight: bold;
}

font-weight: Controla a espessura do texto.

bold: Aumenta a espessura da fonte.

lighter: Diminui a espessura do texto. 

h2 {
        font-style: italic;
}

font-style: Define o estilo do texto. 

h3 {
        color: #fff;
}

color: Define a cor texto.

red: Nome da cor em inglês.

#fff: Código hexadecimal da cor.

rgb(255, 0, 0): Código RGB da cor. 

p {
        text-align: center;
}

text-align: Alinha o texto.

left: Alinha o texto a esquerda.

right: Alinha o texto a direita.

center: Alinha o texto ao centro. 

p {
    text-decoration: line-through;
}

text-decoration: Define a decoração do texto.

none: Valor padrão.

underline: Adiciona uma linha de sublinhado sob o texto.

overline: Adiciona uma linha sobre o texto.

line-through: Adiciona uma linha através do texto, criando um efeito de "riscado". 

h5 {
    text-transform: uppercase;
}

text-transform: Define a transformação do texto.

none: Valor padrão.

uppercase: Transforma o texto em Maiúsculo.

lowercasse: Transforma o texto em Minústuco.

BACKGROUND

Em CSS, o background é uma propriedade que permite controlar o plano de fundo de um elemento HTML.

Sintaxe

elemento {
        background-color: #000;
}

Essa tag do css coloca uma cor no fundo

red: Nome da cor em inglês.

#000: Código hexadecimal da cor.

rgb(255, 0, 0): Código RGB da cor. 

elemento {
        background-image: url('sua-imagem.png');
}

Use esta subpropriedade para definir uma imagem de fundo para o elemento. Você deve fornecer a URL da imagem que deseja usar como plano de fundo.

elemento {
        background-repeat: no-repeat;
}

Essa subpropriedade determina como a imagem de fundo é repetida ou não. Valores comuns incluem repeat, no-repeat, repeat-x e repeat-y.

elemento {
        background-size: contain;
}

Essa subpropriedade controla o tamanho da imagem de fundo. Você pode definir dimensões específicas, usar palavras-chave como cover ou contain, ou especificar porcentagens.

elemento {
        background-position: center top;
}

Define a posição inicial da imagem de fundo. Você pode usar palavras-chave como top, center, right, bottom, left, ou valores específicos como porcentagens ou pixels.
