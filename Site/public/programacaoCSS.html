ESTILIZAÇÃO DE TEXTOS

A estilização de textos em CSS envolve a aplicação de estilos a elementos de texto, como títulos, parágrafos, links e outros.

Sintaxe

p {
        font-family: Arial, sans-serif;
}

font-family: Define a fonte a ser usada para o texto. 

h1 {
        font-size: 24px;
}

font-size: Define o tamanho do texto.

24px: Tamanho fixo em pixels.

2rem: Um valor relativo à fonte do elemento raiz. 

strong {
            font-weight: bold;
}

font-weight: Controla a espessura do texto.

bold: Aumenta a espessura da fonte.

lighter: Diminui a espessura do texto. 

h2 {
        font-style: italic;
}

font-style: Define o estilo do texto. 

h3 {
        color: #fff;
}

color: Define a cor texto.

red: Nome da cor em inglês.

#fff: Código hexadecimal da cor.

rgb(255, 0, 0): Código RGB da cor. 

p {
        text-align: center;
}

text-align: Alinha o texto.

left: Alinha o texto a esquerda.

right: Alinha o texto a direita.

center: Alinha o texto ao centro. 

p {
    text-decoration: line-through;
}

text-decoration: Define a decoração do texto.

none: Valor padrão.

underline: Adiciona uma linha de sublinhado sob o texto.

overline: Adiciona uma linha sobre o texto.

line-through: Adiciona uma linha através do texto, criando um efeito de "riscado". 

h5 {
    text-transform: uppercase;
}

text-transform: Define a transformação do texto.

none: Valor padrão.

uppercase: Transforma o texto em Maiúsculo.

lowercasse: Transforma o texto em Minústuco.

BACKGROUND

Em CSS, o background é uma propriedade que permite controlar o plano de fundo de um elemento HTML.

Sintaxe

elemento {
        background-color: #000;
}

Essa tag do css coloca uma cor no fundo

red: Nome da cor em inglês.

#000: Código hexadecimal da cor.

rgb(255, 0, 0): Código RGB da cor. 

elemento {
        background-image: url('sua-imagem.png');
}

Use esta subpropriedade para definir uma imagem de fundo para o elemento. Você deve fornecer a URL da imagem que deseja usar como plano de fundo.

elemento {
        background-repeat: no-repeat;
}

Essa subpropriedade determina como a imagem de fundo é repetida ou não. Valores comuns incluem repeat, no-repeat, repeat-x e repeat-y.

elemento {
        background-size: contain;
}

Essa subpropriedade controla o tamanho da imagem de fundo. Você pode definir dimensões específicas, usar palavras-chave como cover ou contain, ou especificar porcentagens.

elemento {
        background-position: center top;
}

Define a posição inicial da imagem de fundo. Você pode usar palavras-chave como top, center, right, bottom, left, ou valores específicos como porcentagens ou pixels.

BORDER

é usada para definir as bordas de um elemento HTML. A propriedade border é uma propriedade abreviada que engloba várias outras propriedades relacionadas às bordas.

Sintaxe

elemento  {
        border:  largura estilo cor;
}

largura É a largura da borda. Pode ser especificada em pixels, em unidades absolutas como em ou rem, ou usando porcentagens.

estilo Define o estilo da borda, como solid (linha contínua), dashed (tracejado), dotted (pontilhado), etc.

cor Especifica a cor da borda. Pode ser um nome de cor, um código hexadecimal, um valor RGB, etc.

elemento  {
        border-width:  2px;
        border-style:  solid;
        border-color:  red;
}

border-width Define a largura da borda.

border-style Define o estilo da borda, como solid (linha contínua), dashed (tracejado), dotted (pontilhado), etc.

cor Especifica a cor da borda. Pode ser um nome de cor, um código hexadecimal, um valor RGB, etc.

DISPLAY

A propriedade display em CSS é fundamental para controlar o layout e a formatação dos elementos HTML em uma página. Ela especifica como um elemento deve ser renderizado no layout do documento.

Sintaxe

elemento  {
        display:  block;
}

block Este valor faz com que o elemento seja exibido como um bloco. Ele começa em uma nova linha e ocupa toda a largura disponível. Altura, margens e preenchimentos podem ser controlados.

elemento  {
        display:  inline;
}

inline Este valor faz com que o elemento seja exibido em linha, sem quebrar para uma nova linha. Ele ocupa apenas o espaço necessário e não aceita largura, altura, margens superiores e inferiores.

elemento  {
        display:  inline-block;
}

inline-block Combina as características de inline e block. O elemento é exibido em linha, mas você pode definir largura, altura e outros estilos de bloco.

elemento  {
        display:  none;
}

none Este valor oculta completamente o elemento. O espaço que o elemento ocuparia ainda é reservado, mas o conteúdo não é renderizado.

.container  {
        display:  flex;
}

flex Permite a criação de um modelo de layout flexível. Os elementos dentro de um contêiner flexível podem ser dispostos de maneira fácil e poderosa.

MARGIN E PADDING

Em CSS, as propriedades margin e padding são usadas para controlar o espaçamento em torno e dentro de elementos HTML,

Sintaxe

elemento  {
        margin:  10px;    /* Todos os lados com margem de 10 pixels */
        margin-left:  10px ;    /* Margem a esquerda de 10px */
        margin-right:  30px ;    /* Margem a direita de 30px */
        margin-top:  60px ;    /* Margem superior de 60px */
        margin-bottom:  20px ;    /* Margem inferior de 20px */
}

margin Define o espaçamento externo de um elemento em relação aos elementos vizinhos. Pode aceitar valores para a margem superior, direita, inferior e esquerda, separados por espaços ou especificados individualmente.

elemento  {
        padding:  10px;    /* Todos os lados com espaçamento interno de 10 pixels */
        padding-left:  10px ;    /* Espaçamento interno a esquerda de 10px */
        padding-right:  30px ;    /* Espaçamento interno a direita de 30px */
        padding-top:  60px ;    /* Espaçamento interno superior de 60px */
        padding-bottom:  20px ;    /* Espaçamento interno inferior de 20px */
}

padding Define o espaçamento interno de um elemento, isto é, o espaço entre a borda do elemento e seu conteúdo. Como margin, padding pode ser especificado para os lados superior, direito, inferior e esquerdo.
